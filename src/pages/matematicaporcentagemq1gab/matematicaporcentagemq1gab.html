Resolução

A questão pergunta quanto é 40% de 800.Podemos resolver pensando em regra de três. 100% está para 800, assim como 40% está para x

100%---800

40%--- x

multiplicando cruzado, temos que 100x=800*40

Após feito o calculo, temos que x = 320 reais

Resposta: letra B
